# Portfolio Website — Mykhailo Dubravskyi

Персональний сайт-портфоліо розробника, побудований на **React** та **Material UI**. Проєкт демонструє навички у C# та Unity розробці, а також досвід створення сучасних веб-інтерфейсів.

---

## 🚀 Про мене

Я пишу код, який не соромно закинути в спільний репозиторій. Моя стихія — це "м'ясо" ігор: складні алгоритми, патерни проєктування та робота з даними.

Мені цікаво брати неочевидні задачі і перетворювати їх на стабільний, оптимізований продукт. Люблю працювати в команді, де кожен знає свою справу, а процеси чітко налагоджені.

---

## 🛠 Технологічний стек

- **Frontend**: React (v18+), React Router v6.
- **Стилізація**: Material UI (MUI) з кастомною темною темою.
- **Анімації**: Framer Motion для плавних переходів сторінок та інтерактивних елементів.
- **Іконки**: Material UI Icons.
- **API**: Open-Meteo для отримання погодних даних у реальному часі.

---

## 🗡️ Реалізовані проєкти

- **Chess Engine (CourseWork)**: Повноцінний шаховий рушій на .NET 8.0. Реалізовано алгоритм Minimax з Alpha-Beta відсіканням, оцінку позиції та підтримку FEN-нотації.
- **Console Mini-RPG**: Покрокова рольова гра на принципах чистого ООП. Включає ієрархію ворогів, систему станів (отруєння, приголомшення) та ігровий цикл боївки.
- **Mobile Game Project (W.I.P.)**: Командна розробка мобільної гри на Unity. Архітектура ігрової логіки на C# та інтеграція піксель-арт асетів.

## ⚙️ Встановлення та запуск

Встановіть залежності:

```

npm install

```

Встановіть бібліотеки для анімацій та іконок:

```

npm install framer-motion @mui/icons-material

```

Запустіть проєкт:

```

npm start

```
